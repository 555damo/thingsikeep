The Kept List

How to update
the site

Two things you can do: edit posts directly, or ask Claude to change the design. Both are covered below — no technical knowledge needed.

01

Adding & editing posts

Go to thekeptlist.com/admin.html. You'll need the GitHub token Damiano gave you — paste it into the token field once and click Save. It stays saved in your browser from then on.

Adding a new post

  1. Click + Add post in the top right
  2. Fill in the form (see field guide below)
  3. Click Save & publish

The site updates in about one minute.

Field guide

Date — when you're adding it (today's date is fine)

Category — Music, Travel, Books, Object, Style, or Food

Card type:

  • Text only — just words, no image. Most common.
  • Colour swatch + text — a decorative coloured block above the text
  • Photo + text — a real photo above the text (see photo instructions below)
  • Pull quote — a single line in italic, good for a memorable quote from a book

Background tone — Default is fine. Deep is slightly darker.

Title line 1 — the main title

Title line 2 — a subtitle, artist name, or place name (shown in italic)

Body text — 2–4 sentences about why this is on the list

External link — a shop or website to link to (optional)

Editing a post

Click Edit next to any post in the list. Change what you like, then click Save & publish.

Deleting a post

Click Edit on the post, then the red Delete post button at the bottom of the form.

Adding a photo to a post

  1. Go to github.com/555damo/thekeptlist
  2. Open the img/ folder
  3. Click Add file → Upload files and upload your photo
  4. Back in the admin, choose Photo + text as the card type
  5. In the Image path field, type: img/your-filename.jpg

02

Changing the design

For changing how the site looks — colours, fonts, layout, spacing — use Claude. You describe what you want in plain English, Claude gives you the code, and you paste it into GitHub.

One-time setup: Claude.ai Project

  1. Go to claude.ai and sign in
  2. Click Projects in the left sidebar → New project
  3. Name it: The Kept List
  4. Click Project instructions and paste the text below
  5. Click Add content → Files and upload index.html, detail.html, and contact.html — ask Damiano to send you these, or download them from github.com/555damo/thekeptlist
You are helping manage The Kept List (thekeptlist.com), a personal curation site. The site is a single HTML file called index.html, plus detail.html for long-form posts and contact.html.

When asked to make a design change:
1. Identify the exact CSS or HTML section to change
2. Show only the changed lines or block — not the whole file — unless the change is small enough to show in full
3. Explain in plain English what to do with the code
4. After showing the code, remind me to go to github.com/555damo/thekeptlist, navigate to the file, click the pencil icon to edit, paste in the changes, and click Commit changes

Keep the existing design — terracotta accent, Cormorant Garamond serif for headings, DM Sans for body, parchment/cream backgrounds. Do not suggest new sections or features unless asked.

Making a design change

Just describe what you want. For example:

  • "Make the site title slightly smaller"
  • "Change the terracotta colour to a dusty rose"
  • "Make the body text a bit bigger and easier to read"
  • "Add more space between the cards"

Claude will show you the code to change. Then:

  1. Go to github.com/555damo/thekeptlist
  2. Click the file (usually index.html)
  3. Click the pencil icon to edit
  4. Find the section Claude highlighted and paste in the change
  5. Click the green Commit changes button

The site updates in about one minute.

Tips

If Claude's response is confusing: "Can you explain that more simply?" or "Show me exactly what to copy and paste."

If a change doesn't look right, take a screenshot and paste it into the chat: "This doesn't look right — can you fix it?"

You don't need to understand the code. Just copy and paste what Claude gives you.
